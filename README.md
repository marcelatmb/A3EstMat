# A3EstMat

## ⚙️ Como Executar o Projeto (VENNGERATOR4000)

O **VENNGERATOR4000** é um projeto de aplicação web estática (Front-end puro) desenvolvido em HTML, CSS e JavaScript. Isso significa que ele **não requer a instalação de nenhum servidor local** (como Node.js, Apache ou Python) para ser executado.

Siga os passos abaixo para rodar o projeto em seu ambiente:

### 1. Download do Código

Existem duas maneiras de obter o código:

* **Clonar o Repositório:**
    ```bash
    git clone [https://github.com/marcelatmb/A3EstMat.git](https://github.com/marcelatmb/A3EstMat.git)
    ```
* **Download Manual:** Baixe o arquivo ZIP do repositório através do botão "Code" no GitHub e descompacte-o em um diretório de sua preferência.

### 2. Execução (Abertura no Navegador)

Após o download, basta abrir o arquivo principal da aplicação:

1.  Navegue até a pasta raiz do projeto (`A3EstMat/`).
2.  Localize o arquivo principal: `venn_diagrama_trabalho.html` (ou `index.html`, se for sua página de *deploy*).
3.  **Clique duas vezes** no arquivo.

O projeto será aberto automaticamente no seu navegador web padrão (Chrome, Firefox, Edge, etc.), e você poderá começar a interagir com a calculadora de Diagramas de Venn imediatamente.
